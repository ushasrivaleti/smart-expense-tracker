# Smart Daily Expense Tracker 🚀

An intelligent, student-focused personal finance manager with predictive analytics and premium visuals.

## Features

- **Intelligent Budget Prediction**: Forecasts your month-end spending based on current daily averages.
- **Visual Analytics**: Interactive Line Charts (Spending Trend) and Doughnut Charts (Category Breakdown).
- **Auto-Categorization**: Special keywords tailored for student life (e.g., Canteen, Metro, Stationary).
- **Savings Goal Tracker**: Real-time monitoring of your financial targets.
- **Premium UI/UX**: Glassmorphism design, custom scrollbars, and non-blocking toast notifications.
- **Secure Local Storage**: All data stays on your device.

## Setup & Use

1.  Open `index.html` in any modern web browser.
2.  Register your profile (stored locally).
3.  Set your Monthly Budget and Savings Goal in the **Budget** tab.
4.  Start logging expenses in the **Add Expense** tab.
5.  Monitor your **Dashboard** for predictive alerts and spending trends.

## Tech Stack

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Premium styling with Glassmorphism.
- **JavaScript (Vanilla)**: Core logic and state management.
- **Chart.js**: Dynamic data visualization.
